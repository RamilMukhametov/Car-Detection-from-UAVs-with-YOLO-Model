# Детекция автомобилей с БПЛА с использованием модели YOLO

## 1. Постановка задачи

### Цель проекта проекта

Обучить модель машинного обучения для детекции легковых автомобилей на борту БПЛА в режиме реального времени

<br>

### Требования предъявляемые к модели машинного обучения

**1. Задача модели мошинного обучения:** Детекция одного класса: Легковые автомобили;

**2. Точность:** Обнаруживать легковые автомобили с точностью не менее 90%;

**3. Скорость:** Работать в режиме реального времени;

**4. Ресурсы:** Модель может работать на оборудовании доступном на борту БПЛА. Максимальная вычеслительная мощность при минимальном весе. Например:Jetson Nano или Jetson Xavier NX;

**5. Энергопотребление:** Для минимального сокращения времени полета БПЛА необходимо минимизировать энергопотребление. (Jetson Nano: 5-10 Вт, Jetson Xavier NX: 10-20 Вт).
